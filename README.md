# ECCT
A 2D kinematic character controller for Unity3D

Features:
>First order control

>Swept collision detection

>Overlap recovery

>Step & slope handling

>Arbitrary up vector

>Substepping

>Ground clamping

Example use in CharacterLocomotion.cs
